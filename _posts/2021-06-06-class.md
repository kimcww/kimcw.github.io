# Class Method 사용법  
  
 ### getClass()  
    Class 로 가져오기  
    
 ### forName(str)  
    str 기반으로 class instance 를 가져오기  
 
 ### getConstructor()  
    모든 생성자를 가져오기  
 
 ### getFields()
    모든 멤버 변수 (필드) 가져오기  
 
 ### getMethods()
    모든 메서드 가져오기
