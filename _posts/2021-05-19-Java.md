JDK (Java Develop Kit ) : JRE + 개발을 위한 도구들이 포함 

컴파일러, 디버그 도구 등이 JDK 에 포함 되어 있음  

  
JRE ( Java Runtime Environment ) : JVM + Java 실행에 필요한 라이브러리  
  
JVM ( Java Vitual MAchine ) : Java 컴파일 후에 바이트 코드를 각 OS 별로 해석해주는 기능을 함
