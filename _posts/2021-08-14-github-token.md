# Github Token 문제

    Please use a personal access token instead 라고 뜬다
    
git push 를 하다가 Token 문제로 Fail 이 떴다
Github가 PW 대신 Token을 사용한다고 한다
Github 공식 홈페이지에도 PW 대신 Token 을 사용해달라고 공지되어 있다
